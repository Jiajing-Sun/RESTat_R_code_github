# Contaminated-Training Simulation Code

This folder contains current scripts for the mild training-contamination simulation exercises.

- Driver scripts are in this folder.
- `R/` contains the contaminated-training-specific modules.

Generated outputs and paper tables/figures are not included.

The scripts use the same public critical-value lookup convention as the main simulations and expect generated
critical-value CSV files under `../../critical_values/*/outputs/` unless equivalent local files are supplied.
