This code keeps the global simulation controls aligned with the earlier critical-value
scripts for the main paper: q-grid, T-grid, alpha levels, replications, finite
monitoring grid, and open-end training grid.

The only new user choices are detector-specific tuning parameters that are unavoidable
for alternative detector shapes:

- MOSUM bandwidths h
- weighted-CUSUM length-weight omega
- multiscale MOSUM bandwidth sets and scale weights

The code is exact for Full-CUSUM and for MOSUM / multiscale MOSUM conditional on the
chosen h-grid. Page-CUSUM and weighted-CUSUM are exact only if you request a full scan;
by default they use a geometric length grid to remain computationally feasible when
`grid_monitor = 10000`.
