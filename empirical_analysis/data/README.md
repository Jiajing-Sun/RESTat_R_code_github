# Empirical Data Placeholder

Place the licensed TickData file here as:

```text
SPX.csv
```

The file is not included in this repository. Generated empirical caches such as `empirical_main_results.csv` and
`empirical_alt_results.csv` are also excluded. If those cached paper-output files are absent, the empirical refresh
driver skips the cache-rebuild step and reports the skipped files.
